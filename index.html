HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The h element is for headers, and must be numbered sequentially
  • The ul element is for an unordered list
  • The p element is a paragraph or block of text
  • The li element is a list item
  • The src attribute defines location(s) of images or files
  • The id attribute identifies elements & selects them
  • The link element connects to an external file.
  • The rel attribute specs the relationship between the current doc and the linked doc.
  • The href attribute specs the URL of the external resource.
  • The class attribute specifies one or more class names for an element· Classes are used by CSS and JavaScript to select and access specific elements. The class name is case sensitive.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A period preceeding a card designates it as a class selector.
  • Width, margin, border and box-shadow define the various attributes of a card.
